### WhatsApp MD User Bot

A simple WhatsApp User bot.

## Setup

1. Deploy on Heroku
   - Click [SCAN](https://qr-hazel-alpha.vercel.app/md) and scan the QR code through the "WhatsApp Linked Devices" option in your WhatsApp app.
   - You will get a session ID in WhatsApp, copy the ID only.
   - If you don't have an account on [Heroku](https://signup.heroku.com/), [create an account now](https://signup.heroku.com/).
   - If you don't have a GitHub account, [sign up](https://github.com/join) now.
   - [FORK](https://github.com/Temitopeareo/abromd/fork) this repository.
   - Now [DEPLOY](https://dashboard.heroku.com/new?template=https://github.com/Temitopeareo/abromd).
